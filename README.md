# 📊 SQL Sales Management Assignment

## 🔍 Overview

This project demonstrates SQL skills from basic to advanced level using a Sales Management dataset.

---

## 🧩 Concepts Covered

* Basic Queries (Filtering, Sorting)
* Joins & Aggregations
* Subqueries
* Common Table Expressions (CTE)
* User Defined Functions (UDF)
* Stored Procedures
* Window Functions
* Data Modification

---

## 📁 Files Included

* sales_management_queries.sql → All SQL queries
* Assignment.pdf → Problem statement
* CSV files → Rawdata

---

## 🛠 Tools Used

* SQL Server

---

## 🎯 Key Learnings

* Writing optimized SQL queries
* Performing advanced analytics using window functions
* Building reusable logic using UDF & stored procedures
* Handling real-world business scenarios
